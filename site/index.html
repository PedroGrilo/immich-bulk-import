Why it exists

The Immich CLI, without the command line

Everything the official immich upload supports — exposed as a clean, native macOS app.

No terminal required

Pick folders and photos with native dialogs, flip toggles, and hit start. That's it.

Zero dependencies

The Immich CLI is bundled inside the app and runs on Electron's runtime — no Node or npm to install.

Every CLI flag

Recursive, albums, ignore patterns, concurrency, watch mode, dry run — all one click away.

Live progress & logs

Real-time progress bar, ETA, and a streaming log — plus a preview of the exact command being run.

Secure by design

Your API key is passed via environment variable (never on the command line) and stored encrypted at rest.

Liquid-glass, theme-aware

A polished frosted-glass interface that follows your macOS light or dark appearance automatically.

Getting started

From download to done in five steps

A built-in tutorial walks you through it the first time you open the app.

Create an API key

In Immich: Account Settings → API Keys.

Connect

Enter your server URL and key. It's verified before saving.

Add photos

Queue whole folders or individual files.

Choose options

Albums, recursive, dry run — or leave the defaults.

Upload

Watch live progress and logs. Cancel anytime.

Bring your library home.

Free and open source. Download the latest build, or explore the code on GitHub.
